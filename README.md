# Projeto da Aula 01

## Explicação do Projeto
Este projeto é um sistema de controle de missão que une programação técnica com uma análise ética da ciência. Ele começa lendo dados de telemetria (como temperatura e pressão) de um arquivo JSON para decidir automaticamente se um foguete pode decolar com segurança. Além da checagem básica, o código calcula a autonomia de energia e usa uma lógica de "IA" para classificar riscos e detectar anomalias nos sistemas. O objetivo técnico é tratar dados brutos e transformá-los em decisões críticas de engenharia. Por fim, o projeto inclui uma reflexão sobre a responsabilidade social, defendendo que o avanço espacial deve ser sustentável e ajudar a resolver problemas na Terra, não apenas buscar recordes. É uma demonstração completa de como o código, a física e a ética trabalham juntos em um projeto de alta complexidade.
